# Emojik — Customizable Emoji Picker

**Emojik** is a modern, dependency-free emoji picker inspired by **WhatsApp Web**.  
It features smooth category scrolling, persistent skin tones, automatic recents, local caching, and full JSON configurability.

---

## Features

- Fully standalone — no external dependencies  
- JSON configuration for categories and emojis  
- Local caching of [emojibase-data](https://github.com/milesj/emojibase) with auto-expiration  
- Full skin tone support with persistent user preference  
- Fast emoji search (via *label* or character)  
- Automatic “Recents” section  
- Customizable font (e.g., Google’s *Noto Color Emoji*)  
- Category auto-highlighting while scrolling (like WhatsApp Web)  
- Simple translation support (i18n ready)  
- Optimized rendering for smooth navigation

---

## Installation

Include the script directly in your HTML:

```html
<script src="emojik.min.js"></script>
```
OR
```html
<script src="https://cdn.jsdelivr.net/gh/jeankassio/Emojik@0.9.2/emojik.min.js"></script>
```

## Basic Usage

```html
<div id="emojiPicker"></div>
<textarea id="msg"></textarea>

<script>
const picker = new Emojik({
  container: '#emojiPicker',
  textarea: '#msg'
});
</script>
```

## Default Options

```js
const picker = new Emojik({
  emojibase: 'https://cdn.jsdelivr.net/npm/emojibase-data@latest/en/data.json',
  container: null,
  textarea: null,
  fileconfig: null,
  config: null, //See Below
  width: '100%',
  height: '100%',
  fontFamily: '"Noto Color Emoji", "Segoe UI Emoji", system-ui',
  fonturl: 'https://fonts.googleapis.com/css2?family=Noto+Color+Emoji',
  i18n: {
    search: "Search...",
    recent: "Recent"
  }
});
```

## Options Value

| Option | Default Value | Description |
| -------- | -------- | -------- |
| emojibase | https://cdn.jsdelivr.net/npm/emojibase-data@latest/en/data.json | Link to your favorite emoji base. [See the i18n list here](https://www.jsdelivr.com/package/npm/emojibase-data?tab=files). |
| container | null | CSS selector for the DOM container to attach Emojik |
| textarea | null | CSS selector for the target textarea to insert emojis |
| fileconfig | null | Path to a custom JSON config file |
| config | js object | Directly provide a config array (bypasses fetch) |
| fontFamily | "Noto Color Emoji", "Segoe UI Emoji", system-ui | Emoji display font |
| fonturl | https://fonts.googleapis.com/css2?family=Noto+Color+Emoji | Stylesheet to inject the font |
| width | '100%' | Picker width |
| height | '100%' | Picker height |
| i18n | js Object | Strings to translate Emojik |

## Persistent Storage

- Recents and Skin tone are saved in localStorage
- Emojibase cache is stored per URL with a TTL of 7 days

## Events

#### Emojik dispatches a custom event when an emoji is selected:

```js
picker.root.addEventListener('emojik.click', e => {
  console.log(e.detail.emoji, e.detail.hex);
});
```
